Guides

How to type emoji, symbols, and GIFs on a Mac. Most of these questions have a stock-macOS answer, and each guide leads with it.

Where Mojito shows up, it's because one shortcode beats memorizing an Option-key chord: the app watches for a colon and autocompletes as you type, in any app. Three triggers cover everything:

TypeWhat it searchesExample
:Emoji, by shortcode or keyword:tada → 🎉
::Symbols — every glyph macOS can draw::lambda → λ
:::GIFs, via Giphy, in any app:::excited → GIF panel
