How to type ⌘, ⌥, ⇧ and other Mac key symbols

Handy for writing documentation, shortcut lists, or tutorials that need the real glyphs. None of them have a typing shortcut — here's what each one means, and the fastest way to get it on the page.

::cmd → ⌘

No built-in keystroke types ⌘ — this is Mojito's shortcode. The stock routes are copy-paste and Unicode Hex Input, below.

How do I type the command symbol on a Mac?

There's no keyboard shortcut for ⌘ — macOS never assigned one. The fastest routes are copying it from a reference table like the one below, enabling Unicode Hex Input and holding Option while typing 2318, or typing ::cmd in Mojito. All three land the actual character, U+2318.

The Character Viewer route (and why it's annoying here)

Open the Character Viewer with ⌃ + ⌘ + Space and you can find all ten glyphs, but the search terms aren't what you'd guess. ⌘ is indexed under "place of interest sign," and ⌥ is filed as "option key" rather than anything resembling its name in most style guides. Searching "command symbol" or "command key" turns up nothing. It works once you know the trick, but it's not discoverable.

Unicode Hex Input

If you're typing these often enough to want a real shortcut, enable the Unicode Hex Input source: System Settings → Keyboard → Input Sources → add "Unicode Hex Input." Switch to it, then hold Option and type the four-digit code from the table above — Option + 2318 types ⌘. It's fast once set up, but it's a dedicated input source you have to switch into and out of, which is friction of its own if you only need these glyphs occasionally.

Faster with Mojito

If you write documentation or shortcut guides more than once a week, Mojito is the least fiddly option of the three. Type ::cmd (or ::command), ::option, ::shift, ::control, ::tab, ::escape, ::return, ::delete, ::capslock, or ::eject anywhere and it autocompletes the right glyph at your cursor — no input source to switch, no obscure Character Viewer search term to remember.

Where these glyphs come from

What does ⌥ mean on a Mac keyboard?

It represents the Option key, sometimes labeled "alt" on the physical key itself (both print on most Mac keyboards). In documentation and menus, ⌥ and "Option" refer to the same key.

Why is ⌘ called the place of interest sign?

Designer Susan Kare needed a symbol for the Command key that wasn't the Apple logo, and found a looped square glyph in an international symbol dictionary — used on Swedish road signs and campground maps to mark a place of cultural interest. It stuck, and the Unicode standard still calls U+2318 the "place of interest sign" today.

How do I show keyboard shortcuts in Markdown or HTML?

Wrap each key in a <kbd> tag: <kbd>⌘</kbd> + <kbd>C</kbd>. Most Markdown renderers pass raw HTML through, so this works in both.
