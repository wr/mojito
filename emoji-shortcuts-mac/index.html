Emoji shortcuts on Mac

macOS has more emoji shortcuts than most people know about. Here's every one of them, from the built-in picker to typing :tada: and getting 🎉 in any app.

⌃⌘Space — the built-in emoji picker

Press Control + Command + Space in any text field and macOS opens its emoji picker next to your cursor. Type to search, click or press Return to insert. This is the shortcut Apple wants you to memorize, and it works everywhere.

The catch is the round trip. The picker is a separate panel, so every emoji means opening it, searching, clicking, and getting back to what you were writing.

The Globe key

On Macs from 2020 on, the fn key doubles as 🌐. Open System Settings → Keyboard and set "Press 🌐 key to" → Show Emoji & Symbols. One tap now opens the same picker as ⌃⌘Space.

Text replacements

Under System Settings → Keyboard → Text Replacements you can map any string to an emoji — omw to 🏃, say. It syncs through iCloud to your iPhone too. It works well for a handful of favorites, and poorly as a way to reach all 3,800-ish emoji: every mapping is one more thing to define by hand.

Type :shortcodes: anywhere

Slack, Discord, and GitHub trained everyone to type :fire: and get 🔥. macOS doesn't do that natively — that's the gap Mojito fills. It's a small free menu-bar app: type a colon and a few letters in any app, an autocomplete appears at your cursor, Return inserts the emoji. It searches by keyword as well as by code (:happy finds 😀), does symbols (::lambda → λ) and GIFs (:::), and stays out of apps that already handle shortcodes.

Which shortcut should you use?

What's the fastest way to type emoji on a Mac?

For an emoji you use constantly, a text replacement is instant. For everything else, an inline shortcode picker like Mojito is faster than the built-in panel because your hands never leave the keyboard and your eyes never leave the line you're writing.

Is there a keyboard shortcut without any setup?

Yes — ⌃⌘Space works out of the box on every Mac. Setup only buys you speed: the Globe key saves a chord, and shortcodes remove the panel round-trip entirely.

Can I change what ⌃⌘Space opens?

Mojito can take over that shortcut (and the Globe key) so it opens a searchable emoji browser instead of the macOS panel. It's an option in Settings → General.
