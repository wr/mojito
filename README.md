# Mojito — marketing site + update feed

This branch (`gh-pages`) is served at <https://mojito.wells.ee>.

- `index.html`, `style.css`, `picker.js` — the marketing page
- `appcast.xml` — Sparkle update feed, written by `scripts/release.sh` on `main`
- `CNAME` — custom domain for GitHub Pages

Source for the app itself is on the `main` branch.
